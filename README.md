# React-Native-Book-Explorer-App
React-based web application that allows users to search, browse, and explore books using an external API. Built with React, JavaScript, and CSS, it provides a simple, interactive interface for discovering books.

**Features:**

Search Books – Users can search for books by title, author, or keyword.

Book List Display – Shows search results with book cover, title, and author.

Responsive Design – Works on desktop and mobile screens.

API Integration – Fetches real-time data from an external books API.

Interactive UI – Clean layout with smooth user interactions.

**Tech Stack:**

Frontend: React.js, HTML, CSS

API: Open Library API

State Management: React Hooks (useState, useEffect)

Package Manager: npm

**Note:**

This project rar file was split into multiple parts because of GitHub’s file size limit.

Requirements: Bash or Any Terminal that support cat command.

**Step1:** Download the spit parts. Make sure the file name are same. Example: file_part_aa, file_part_ab.

**step2:** Run this command : cat file_part_* > book-explorer.rar

**step3:** Extract the Project. Command :unzip book-explorer.rar

**How to Run This Project on Your System:**

1. Make sure you have the following installed:

Node.js (v16 or later)
npm (comes with Node.js)
A web browser.

2. Download the files (All parts and combine it using the steps above) and extract it.

3. Open the Folder in VS Code and then open the terminal.
   
4. Make sure you are in the folder book-explorer. Then type command npm install then npm start After that press w to open Web Browser
